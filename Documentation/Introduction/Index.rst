What does it do?
================

With `mysqlreport` you can analyze your MySQL/MariaDB server. It will show you
some hints about InnoDB usage, table cache, thread cache, uptime and an overall
overview. Additional, you can activate profiling of each query in extension
settings and show details of all queries of a TYPO3 request in BE module.
